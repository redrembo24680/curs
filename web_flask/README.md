# Flask фронтенд для системи голосування

Веб-інтерфейс створений на Flask. Застосунок звертається до C++ REST API (`server_cpp`) та дозволяє:

- переглядати матчі й гравців;
- віддавати голоси в реальному часі;
- створювати нових гравців та матчі;
- переглядати статистику та стан API.

## Запуск

```bash
cd web_flask
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
flask --app app run
```

> API базова адреса змінюється через змінну `API_BASE_URL` (за замовчуванням: `http://localhost:8080/api`).

## Структура

```
web_flask/
├── app.py              # Точки входу Flask
├── start.py            # Автоматизований запуск
├── requirements.txt    # Python залежності
├── templates/          # Jinja2 шаблони
├── static/style.css    # Стилі інтерфейсу
└── flask_session/      # Каталог для cookie-сесій
```

## Env змінні

| Назва              | Призначення                           |
|--------------------|----------------------------------------|
| `API_BASE_URL`     | Базова адреса C++ API                 |
| `FLASK_SECRET_KEY` | Ключ для підпису сесій                |
| `WEB_PORT`         | Порт для локального запуску Flask     |

## Сторінки

1. `Головна` – вибір матчу та голосування.
2. `Гравці` – додавання та огляд гравців.
3. `Матчі` – створення та список матчів.
4. `Статистика` – рейтинги гравців і матчів.
5. `Стан API` – перевірка відповіді backend-сервера.
